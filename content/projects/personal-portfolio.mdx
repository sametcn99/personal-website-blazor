# Personal Portfolio Website

A sleek and modern portfolio website showcasing my projects, blog posts, and professional experience.

## Overview

This project is a fully responsive personal portfolio website built using Blazor WebAssembly and styled with MudBlazor components. It features a clean, professional design with smooth animations and an intuitive user interface.

## Features

- **Responsive Design**: Fully responsive layout that works on all devices
- **Dark Mode**: Toggle between light and dark themes
- **Blog Integration**: Markdown-based blog with syntax highlighting
- **Project Showcase**: Display projects with images, descriptions, and links
- **Contact Form**: Integrated contact form with email functionality
- **SEO Optimized**: Meta tags and structured data for better search visibility

## Tech Stack

- **Frontend**: Blazor WebAssembly
- **UI Framework**: MudBlazor
- **Styling**: CSS with custom properties
- **Deployment**: GitHub Pages / Netlify
- **Content**: Markdown files

## Installation

```bash
# Clone the repository
git clone https://github.com/username/portfolio.git

# Navigate to project directory
cd portfolio

# Restore dependencies
dotnet restore

# Run the application
dotnet run
```

## Deployment

The site is automatically deployed to GitHub Pages using GitHub Actions whenever changes are pushed to the main branch.
