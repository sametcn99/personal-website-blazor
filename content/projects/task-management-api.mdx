# Task Management API

A robust RESTful API for managing tasks, projects, and teams built with ASP.NET Core.

## Overview

This project provides a comprehensive API for task management applications. It includes user authentication, role-based authorization, and full CRUD operations for tasks, projects, and teams.

## Features

- **JWT Authentication**: Secure token-based authentication
- **Role-Based Authorization**: Admin, Manager, and User roles
- **CRUD Operations**: Complete task and project management
- **Filtering & Pagination**: Efficient data retrieval
- **Swagger Documentation**: Interactive API documentation
- **Database Migrations**: EF Core migrations for schema management

## Tech Stack

- **Framework**: ASP.NET Core 8.0
- **ORM**: Entity Framework Core
- **Database**: PostgreSQL
- **Authentication**: JWT Bearer tokens
- **Documentation**: Swagger/OpenAPI
- **Testing**: xUnit, Moq

## API Endpoints

### Authentication

```
POST /api/auth/register - Register new user
POST /api/auth/login - Login and get JWT token
POST /api/auth/refresh - Refresh access token
```

### Tasks

```
GET    /api/tasks - Get all tasks (with pagination)
GET    /api/tasks/{id} - Get task by ID
POST   /api/tasks - Create new task
PUT    /api/tasks/{id} - Update task
DELETE /api/tasks/{id} - Delete task
```

### Projects

```
GET    /api/projects - Get all projects
GET    /api/projects/{id} - Get project by ID
POST   /api/projects - Create new project
PUT    /api/projects/{id} - Update project
DELETE /api/projects/{id} - Delete project
```

## Example Usage

```csharp
// Task model
public class Task
{
    public int Id { get; set; }
    public string Title { get; set; }
    public string Description { get; set; }
    public TaskStatus Status { get; set; }
    public DateTime DueDate { get; set; }
    public int ProjectId { get; set; }
    public int AssignedToUserId { get; set; }
}

// Create task endpoint
[HttpPost]
[Authorize]
public async Task<ActionResult<TaskDto>> CreateTask(CreateTaskDto dto)
{
    var task = _mapper.Map<Task>(dto);
    task.CreatedBy = User.GetUserId();
    
    await _context.Tasks.AddAsync(task);
    await _context.SaveChangesAsync();
    
    return CreatedAtAction(nameof(GetTask), 
        new { id = task.Id }, 
        _mapper.Map<TaskDto>(task));
}
```

## Setup

git clone https://github.com/username/task-api.git

# Navigate to project

# Update connection string in appsettings.json
