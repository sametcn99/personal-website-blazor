# Weather Dashboard

An interactive weather dashboard providing real-time weather data, forecasts, and visualizations.

## Overview

This Blazor WebAssembly application displays current weather conditions, hourly and daily forecasts, and interactive weather maps. It integrates with the OpenWeatherMap API to provide accurate and up-to-date weather information.

## Features

- **Current Weather**: Real-time weather conditions for any location
- **Hourly Forecast**: 48-hour detailed forecast
- **Daily Forecast**: 7-day weather outlook
- **Interactive Maps**: Temperature, precipitation, and wind maps
- **Location Search**: Search weather by city name or coordinates
- **Favorite Locations**: Save and quickly access favorite cities
- **Weather Alerts**: Severe weather warnings and notifications
- **Responsive Charts**: Beautiful visualizations using Chart.js

## Tech Stack

- **Frontend**: Blazor WebAssembly
- **UI Components**: MudBlazor
- **Charts**: Blazor.Charts / Chart.js
- **API**: OpenWeatherMap API
- **State Management**: Fluxor
- **HTTP Client**: HttpClient with Polly for resilience

## Key Features Implementation

### Weather Service

```csharp
public class WeatherService : IWeatherService
{
    private readonly HttpClient _httpClient;
    private readonly string _apiKey;

    public async Task<WeatherData> GetCurrentWeatherAsync(string city)
    {
        var response = await _httpClient.GetAsync(
            $"weather?q={city}&appid={_apiKey}&units=metric"
        );
        
        response.EnsureSuccessStatusCode();
        return await response.Content.ReadFromJsonAsync<WeatherData>();
    }

    public async Task<ForecastData> GetForecastAsync(string city)
    {
        var response = await _httpClient.GetAsync(
            $"forecast?q={city}&appid={_apiKey}&units=metric"
        );
        
        response.EnsureSuccessStatusCode();
        return await response.Content.ReadFromJsonAsync<ForecastData>();
    }
}
```

## Installation

```bash
# Clone the repository
git clone https://github.com/username/weather-dashboard.git

# Navigate to project
cd weather-dashboard

# Add your API key to appsettings.json
# "WeatherApi": { "ApiKey": "your-api-key-here" }

# Run the application
dotnet run
```

## Configuration

Get a free API key from [OpenWeatherMap](https://openweathermap.org/api) and add it to your `appsettings.json`:

```json
{
  "WeatherApi": {
    "BaseUrl": "https://api.openweathermap.org/data/2.5/",
    "ApiKey": "your-api-key-here"
  }
}
```
