# JavaScript Array Methods Guide

Master JavaScript array methods with these practical examples.

## Iteration Methods

```javascript
// forEach - Execute function for each element
numbers.forEach(num => console.log(num));

// map - Transform each element
const doubled = numbers.map(num => num * 2);

// filter - Keep elements that pass test
const evens = numbers.filter(num => num % 2 === 0);

// reduce - Reduce to single value
const sum = numbers.reduce((acc, num) => acc + num, 0);
```

## Finding Elements

```javascript
// find - Get first matching element
const user = users.find(u => u.id === 1);

// findIndex - Get index of first match
const index = users.findIndex(u => u.id === 1);

// some - Check if any match
const hasAdults = users.some(u => u.age >= 18);

// every - Check if all match
const allAdults = users.every(u => u.age >= 18);

// includes - Check if array contains value
const hasApple = fruits.includes('apple');
```

## Adding/Removing Elements

```javascript
// push - Add to end
fruits.push('orange');

// pop - Remove from end
const last = fruits.pop();

// unshift - Add to beginning
fruits.unshift('apple');

// shift - Remove from beginning
const first = fruits.shift();

// splice - Add/remove at position
fruits.splice(2, 1, 'banana'); // Remove 1 at index 2, add 'banana'
```

## Combining Arrays

```javascript
// concat - Combine arrays
const combined = arr1.concat(arr2);

// spread operator
const combined = [...arr1, ...arr2];

// flat - Flatten nested arrays
const flattened = [[1, 2], [3, 4]].flat();

// flatMap - Map then flatten
const result = arr.flatMap(x => [x, x * 2]);
```

## Sorting and Reversing

```javascript
// sort - Sort array
numbers.sort((a, b) => a - b); // Ascending
numbers.sort((a, b) => b - a); // Descending

// reverse - Reverse array
const reversed = numbers.reverse();
```

## Creating Arrays

```javascript
// Array.from - Create from iterable
const arr = Array.from('hello'); // ['h', 'e', 'l', 'l', 'o']
const range = Array.from({ length: 5 }, (_, i) => i); // [0, 1, 2, 3, 4]

// Array.of - Create from arguments
const arr = Array.of(1, 2, 3); // [1, 2, 3]

// fill - Fill with value
const zeros = new Array(5).fill(0); // [0, 0, 0, 0, 0]
```

## Slicing

```javascript
// slice - Extract portion
const portion = fruits.slice(1, 3); // Elements at index 1 and 2

// Copy array
const copy = fruits.slice();
```

## Advanced Examples

```javascript
// Chain methods
const result = users
  .filter(u => u.active)
  .map(u => u.name)
  .sort();

// Group by property
const grouped = users.reduce((acc, user) => {
  const key = user.role;
  if (!acc[key]) acc[key] = [];
  acc[key].push(user);
  return acc;
}, {});

// Remove duplicates
const unique = [...new Set(numbers)];
```
